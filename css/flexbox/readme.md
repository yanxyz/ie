# Flexbox in IE

IE10 开始支持 flexbox，需要加前缀 `-ms-`。
IE11 不需要前缀。

flexbox 语法至今有数次改动，IE10 的 flexbox 语法已经过时。
比如，IE10 flex 默认值是 none。
